                                                                      Sayısal Loto Uygulaması

Python öğrenme sürecimde, temel konuları pekiştirmek için geliştirdiğim basit bir şans oyunları simülasyonu.


Class Yapısı: Kodun daha derli toplu durması için nesne yönelimli programlama (OOP) mantığıyla çalıştım.
Random Modülü: Rastgele sayı üretimi için random kütüphanesini kullandım.
Kontrol Mekanizması: Aynı kolon içinde aynı sayının tekrar etmemesi ve kullanıcı hatalı giriş yaptığında programın çökmemesi için basit kontroller ekledim.
Sıralama: Üretilen sayıların daha okunaklı olması için listeleri küçükten büyüye sıraladım.
Derslerde öğrendiğim döngüler (while/for) ve liste metodlarını pratik etmek için güzel bir başlangıç projesi oldu.
